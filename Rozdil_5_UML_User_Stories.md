РОЗДІЛ 5. ВІДОБРАЖЕННЯ USER STORIES У UML-НОТАЦІЇ
5.1 Тип діаграми

Для відображення функціональних сценаріїв використовується UML-діаграма варіантів використання (Use Case Diagram).

5.2 Основні актори системи
Адміністратор
Страховий агент
Менеджер
Клієнт

5.3 Основні варіанти використання
Для адміністратора:
Управління користувачами
Налаштування ролей
Перегляд журналів
Для страхового агента:
Реєстрація клієнта
Створення договору
Реєстрація страхового випадку

Для менеджера:
Перегляд звітів
Аналіз статистики

Для клієнта:
Перегляд договору
Подання заявки на страховий випадок
Перевірка статусу виплати
